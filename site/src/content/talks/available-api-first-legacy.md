## Workshop Overview

This interactive workshop uses the "APIs for Lemonade Stands" methodology to teach API-first design thinking in an accessible, memorable way.

## What Participants Will Learn

- The API-first mindset and why it matters for legacy modernization
- How to identify resources and design clean API boundaries
- Contract-first development practices
- Strategies for incremental migration from legacy to modern architectures

## Ideal For

- Technical architects and senior developers
- Engineering leaders planning modernization initiatives
- Teams transitioning from monolithic to service-oriented architectures
