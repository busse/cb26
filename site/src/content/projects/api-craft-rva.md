I founded **API Craft RVA** in 2014 to build Richmond's API-focused developer community. Over six years, we grew to 288+ members and hosted 13+ events covering API design, governance, strategy, and best practices.

## Mission

The meetup brought together the Richmond tech community to discuss:

- **API Design** – Best practices for RESTful API design
- **API Strategy** – How organizations can leverage APIs for business value
- **API Governance** – Managing APIs at enterprise scale
- **Developer Experience** – Making APIs a joy to use

## Notable Events

- **January 17, 2017:** "Capital One DevExchange Open Discussion & Feedback"
- **October 25, 2016:** "The 3 Ways Melvin Conway Saved Our Enterprise API Governance"
- **March 25, 2020:** "Providing Business Value with Observability" (via Zoom)
