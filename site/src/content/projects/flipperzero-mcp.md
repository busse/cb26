An unofficial **Model Context Protocol (MCP)** server for the [Flipper Zero](https://flipperzero.one/), the popular portable multi-tool for hardware hackers.

## What It Does

flipperzero-mcp bridges the gap between AI assistants and physical hardware, allowing Claude and other MCP-compatible AI tools to:

- Interact with Flipper Zero devices
- Query device state and capabilities
- Control device functions programmatically

## Why MCP?

The Model Context Protocol is Anthropic's open standard for connecting AI assistants to external tools and data sources. This project demonstrates how MCP can extend AI capabilities into the physical world of hardware hacking and IoT.

## Part of the MCP Ecosystem

This is one of several MCP-related projects exploring how AI assistants can interface with the real world beyond text and code.
