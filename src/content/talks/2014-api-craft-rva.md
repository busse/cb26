Founded the API Craft RVA Meetup in 2014 to build a local community for API practitioners in Richmond, Virginia. Over six years, organized and hosted 13 events, growing the group to 288 members.

## Topics Covered Across the Series

- API design principles and best practices
- Developer experience and documentation
- Contract-first and design-first development
- API strategy for organizations large and small
- RESTful architecture and hypermedia
- Real-world case studies from local practitioners

The meetup brought together developers, architects, product managers, and technology leaders across Richmond's tech community to share knowledge and trade stories about building and consuming APIs.
