Software engineering in 2026 increasingly looks like directing AI agents rather than writing every line yourself. But doing it well requires new skills: knowing how to decompose problems, when to trust agent output, and how to build workflows that actually ship.

In this workshop, we'll roll up our sleeves with Claude Code, Anthropic's terminal-native agentic coding tool. You'll work through hands-on exercises covering codebase navigation, task orchestration, prompt structuring, and debugging agent output.

For those ready to go deeper, we'll explore multi-agent patterns: running parallel Claude sessions, coordinating agent teams that research, build, and verify concurrently, and building subagent pipelines for code review and testing.

No Claude Code experience needed, just bring a laptop with a terminal.
