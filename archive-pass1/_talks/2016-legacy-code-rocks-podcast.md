Joined Andrea Goulet and M. Scott Ford on the Legacy Code Rocks podcast to discuss APIs and digital transformation.

## Topics Discussed

- **What is an API?** – Breaking down the fundamentals for a broader audience
- **What is critical to API success?** – Beyond technology to strategy and culture
- **How to find strategic success with digital initiatives** – Lessons from enterprise API programs
- **Experience launching the first public API platform for a North American bank** – The Capital One DevExchange story

## About Legacy Code Rocks

Legacy Code Rocks explores how to work with existing codebases, modernize systems, and navigate the challenges of maintaining and improving software over time. This episode focused on how APIs can be a powerful tool for modernizing legacy systems.
